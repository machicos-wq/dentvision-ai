# DentVision AI PWA

Versione installabile come web app.

## Come provarla su Android
1. Estrai lo ZIP.
2. Apri `index.html` con Chrome.
3. Tocca i tre puntini.
4. Premi "Aggiungi a schermata Home".

## Cosa fa
- Stima danni grandine
- Anteprima foto
- Salvataggio richieste nel browser
- Copia testo
- Invio richiesta su WhatsApp
- Manifest PWA + service worker

## Limite attuale
Aprendola da file locale alcune funzioni PWA possono essere limitate. Per installarla bene va messa online su Netlify, Vercel, GitHub Pages o server proprio.
