# DentVision AI v4.4 - Risultati ordinati

Versione index unico.

## Migliorie principali

- Niente più elenco infinito di bollature sotto la foto.
- Risultati raggruppati per pannello.
- Totale compatto: facili, medi, difficili.
- Dettaglio tecnico opzionale e chiuso di default.
- I numeri restano sulla foto, mentre la pratica usa un riepilogo pulito per preventivo e report.
- Limite rilevamenti foto ridotto a 70 per evitare liste ingestibili e falsi positivi esagerati.

## Caricamento

Sostituire solo `index.html` nel repository GitHub.

Commit consigliato:

`DentVision AI v4.4 - risultati ordinati e riepilogo pannelli`

Aprire il sito con:
